# Project Structure

Use this map to quickly find files:

- `factor_analysis_MAPLE.ipynb`  
  Main notebook.

- `maple_kalman_valuation.py`  
  Standalone valuation script.

- `maple_factor_analysis_charts.py`  
  Standalone factor-chart script.

- `charts/data/`  
  Source CSV inputs (Maple, Aave, Morpho, BTC, ETH, benchmark files).

- `charts/figures/`  
  All generated PNG charts (factor, valuation, break tests, dashboard visuals).

- `outputs/tables/`  
  Generated CSV outputs (peer multiples, fee momentum summaries).

- `outputs/summaries/`  
  Generated markdown summaries.

- `notebooks/reference/`  
  Reference notebook/materials moved out of `charts/`.
